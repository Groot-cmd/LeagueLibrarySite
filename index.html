Before starting

For developing your app with this library, make sure you read and understand Riot Games terms and conditions also read the developer portal.


The basic

First we will instantiate the Host class and store it in a variable which in our case we will use the host variable:

host = LeagueLibrary.host.Host('platformID', 'region','Token') the variable is very important as we will use it later.

The platformID parameter accepts the following values, ["br1","eun1", "euw1","jp1","kr","la1", "la2","na1","oc1","tr1", "ru"].

The regions parameter accepts the following values, ["americas","asia","europe"].

The Token parameter is your api key obtained through the official Riot Games API website.

Now we will instantiate the Summoner class and store it in a variable that in our case we will use the summoner variable and pass the host variable as a parameter. we stated previously:

summoner = LeagueLibrary.summoner.Summoner(host) The variable is very important we will use later.

Currently the Summoner class has only 4 (four methods) being them, byEncryptedAccountId, bySummonerName, byEncryptedPUUID and byEncryptedPUUID, in our case we will use, bySummonerName passing as parameter a valid invoker name on the previously chosen server:

summoner.bySummonerName('summonerName').

In this case, it is not necessary to store it in any variable, as we can access the values ​​returned in an internal variable of the byNameJson class.

To get the values ​​we will use the class variable which in our case is summoner followed by the name of the internal variable byNameJson, with Python's print() function: print(summoner.byNameJson) will print all values ​​in a dictionary in the terminal.

We can also store the values ​​returned from the method in a variable:

summoner1 = summoner.bySummonerName('summonerName')

Method called and stored in the variable summoner1

print(summoner1) will return the full dictionary

print(summoner1['id']) will return the value of the key passed.
